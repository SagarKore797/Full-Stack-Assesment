# Full-Stack-Assesment

Steps to be followed 
Repo Structure:
- backend (Java, Spring Boot)
- frontend (React)

Setup Steps:
1. Backend:
   - Open `backend` folder
   - Create schema (update schema name in `application.properties`)
   - Run: `mvn spring-boot:run`
2. Frontend:
   - Open `frontend` folder in VS Code (or preferred editor)
   - Run: `npm install`
   - Start dev: `npm run dev`
